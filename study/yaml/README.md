## Setup

```sh
git clone https://github.com/jbeder/yaml-cpp.git
```

```sh
cd yaml-cpp
```

```sh
mkdir build
```

```sh
cd build
```

```sh
cmake ..
```

```sh
cmake --build .
```

```sh
sudo cmake --install .
```

## Build

```sh
make
```

## Test

```sh
./main -c config.yml
```
